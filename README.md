# Parchate - Gestión Emocional con IA

Aplicación para procesar emociones y pensamientos mediante diario y chat con IA.

## Características
- Diario emocional
- Chat terapéutico con IA
- Seguimiento de estado de ánimo
- Análisis de patrones

## Tecnologías
- Frontend: React + Tailwind CSS
- Backend: Flask + SQLite
- IA: OpenAI / Hugging Face
